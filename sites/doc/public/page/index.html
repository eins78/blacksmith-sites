How To Make A Page

by nodejitsu on 2011-11-03T18:55:54.294Z
page

Each article requires it's own folder in the articles folder with two important files:

articles/:my-article/article.md
articles/:my-article/article.json

article.md

The articles themselves are parsed as github-flavored markdown, which is arguably the best markdown dialect around. Just write your article content in this file as markdown and save it.

For example:

 ## This is my Blost Post
 It's **awesome**
 *Super Awesome*

article.json

Each article has an associated metadata file found in article.json.

For example:

{
  "title": "My First Blog Post",
  "author": "Joshua Holbrook",
  "date": "Fri Aug 19 17:36:01 PDT 2011"
}

The three important fields are title, author and date

Each folder should have the following files in it:

  • page.json
  • content.md
