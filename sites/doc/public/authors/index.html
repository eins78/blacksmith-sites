Managing Authors

by nodejitsu on 2011-11-03T18:55:54.294Z
authors

Authors specified in the article.json are cross-referenced against the files in the /authors folder.

For instance, the value of "Joshua Holbrook" in the author field of article.json will load the data from: /authors/Joshua Holbrook.json:

Sample Authors File

{
  "email": "<a href="mailto:josh.holbrook@gmail.com"josh.holbrook@gmail.com</a",
  "github": "jesusabdullah",
  "twitter": "jesusabdullah",
  "location": "Oakland, CA"
}

Avatars are powered by Gravatar, using the value you specify in the email field.
